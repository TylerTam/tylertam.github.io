Purpose & Goal


My goal for this level was to design a 4v4 PvP asymmetrical map that would also support PvE gameplay. This meant diving a bit more into encounter design, while also designing the level. This was my first time designing encounters for PvP, and I was looking forward to doing it, and learning a bit more about that area.

This project started out as a hobby project, so I didn't document as much as I normally do. I did keep an LDD up, while creating diagrams, as this was how I generally design levels, but I didn't keep much documentation on my progress.
