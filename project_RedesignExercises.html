Redesign Exercises


Redesign Exercises

An exercise I like to do is take existing systems, mechanics, game modes, from games that I have played, and not necessarily worked on, and try to find ways to improve them. I found that doing this helps me exercise my design skills and improve upon them.

Note: I did not personally work on any of these games when redesigning them.


Halo Infinite


Stockpile Redesign


Stockpile is a game mode in Halo Infinite that I found that players generally had low enjoyment while playing it. I designed some possible changes to the game mode, and also a possible overhaul to the overall design.
